# Venda+ 📈

O **Venda+** é uma aplicação web Full-Stack desenvolvida para facilitar a gestão de vendas e o controle de clientes para vendedores autônomos. O projeto surgiu da necessidade de centralizar informações que, muitas vezes, ficam dispersas em anotações manuais, permitindo uma visão clara do desempenho comercial.

## 🚀 Funcionalidades

* **Gestão de Clientes:** Cadastro completo, edição e exclusão de contatos.
* **Controle de Vendas:** Registro de pedidos vinculados a clientes específicos.
* **Dashboard de Resultados:** Visualização rápida de métricas de vendas (em desenvolvimento).
* **Interface Responsiva:** Adaptada para uso em dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

Este projeto utiliza a stack JavaScript para garantir performance e escalabilidade:

### Frontend
* **Bootstrap:** Framework CSS para garantir um design moderno e responsivo.
* **JavaScript:** Lógica de interação e consumo da API.

### Backend
* **Node.js & Express:** Ambiente de execução e framework para a construção da API REST.
* **MySQL:** Banco de dados relacional para armazenamento seguro dos dados.

## 🔧 Instalação e Configuração

Para rodar o projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/patrickribeiro88/projeto_VendaMais.git](https://github.com/patrickribeiro88/projeto_VendaMais.git)
    ```

2.  **Configuração do Banco de Dados:**
    * Certifique-se de ter o MySQL instalado.
    * Crie um banco de dados para o projeto.
    * Importe o script SQL (caso disponível no repositório) para criar as tabelas necessárias.

3.  **Configuração do Backend:**
    * Acesse a pasta `backend`: `cd backend`
    * Instale as dependências: `npm install`
    * Crie um arquivo `.env` baseado no exemplo do repositório e preencha com as suas credenciais do banco MySQL.
    * Inicie o servidor: `npm start`

4.  **Configuração do Frontend:**
    * Acesse a pasta `frontend`: `cd ../frontend`
    * Instale as dependências: `npm install`
    * Inicie a aplicação: `npm run serve` (ou comando equivalente).
